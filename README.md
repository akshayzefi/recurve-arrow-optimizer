Recurve archery performance is highly dependent on arrow design parameters such as arrow length, point weight, shaft density, and aerodynamic properties. Conventional optimization methods rely heavily on manual
experimentation and trial-and-error processes, which are time-intensive and inefficient. This project proposes a machine learning-based optimization framework to enhance recurve arrow performance by predicting key
performance indicators and identifying optimal configurations. A multi-output regression model based on XGBoost was developed to predict arrow speed and grouping accuracy simultaneously. Bayesian optimization
was employed to search for optimal parameter combinations by maximizing speed and minimizing group size. The results demonstrate that machine learning provides a reliable, efficient, and scalable approach to
engineering design optimization in sports equipment.
